# # BBC News Homepage Layout

This project recreates a BBC-style news homepage using HTML and CSS. The goal was to practice building complex layouts using modern CSS techniques such as Grid, Flexbox, and responsive design.

## Preview

Open the project using Live Server or open `index.html` directly in your browser.


https://yonathanjaramillo.github.io/bbc-css/

## Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Responsive Design

## Features

* BBC-style navigation bar
* Large hero news section
* Image overlays with headlines
* Multi-column news layout
* Responsive grid system
* News cards with images and descriptions

## What I Practiced

* Recreating real-world website layouts
* Using CSS Grid for complex layouts
* Combining Grid and Flexbox
* Overlaying text on images
* Responsive design for different screen sizes
* Structuring semantic HTML content
